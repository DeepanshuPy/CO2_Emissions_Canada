# Canada CO2 Emissions Analysis

Data Wrangling & Visual Storytelling Project

## 📌 Project Overview

This project provides a comprehensive analysis of Greenhouse Gas (GHG) emission trends in Canada. Using Python, I transformed raw government data into a series of visualizations that highlight which provinces and sectors are the primary drivers of emissions and how these patterns have evolved over the last three decades.

Key Objective: To identify the "pivot points" where Canadian climate policy and industrial shifts have most significantly impacted CO2 output.

## 🛠️ Technical Architecture

I followed a modular data science workflow to ensure the analysis is reproducible and scalable:

1. Data Cleaning: Utilizing Pandas to handle missing values, filter specific years, and aggregate data by province.

2. Analysis: Using NumPy for calculating Year-over-Year (YoY) growth rates and percentage contributions.

3. Visualization: Leveraging Seaborn and Matplotlib to create multi-line plots, heatmaps, and stacked bar charts for sectoral comparisons.

4. ## 📊 Data Insights

5. Based on the Data_Analysis_2026.ipynb findings:

1. Provincial Leaders: Identified which provinces (e.g., Alberta, Ontario) contribute the highest share of national emissions.

2. Sectoral Impact: A deep dive into the Transportation and Energy sectors to see their specific contribution to the GHG footprint.

3. Long-term Trends: Visualized Canada's trajectory relative to 1990 baseline levels.
